Typescript with AngularJS and Node in VS Code.

By Alex Earley, Sunday November 19th, 2017

  It took me the better part of yesterday to build this small little 'Hello World' app. I am glad that I did. AngularJS is my bread and butter at work, and my boss has shown interest in typescript. I wanted to introduce TypeScript to my stack, but not do it in the middle of a sprint at work. So I updated VS Code and installed TypeScript via NPM. I Google'd a handful of errors and questions, and soon enough I had a 'Hello World' alert popping up on ng-click.

Why TypeScript?

  TypeScript is a superset of javascript. It compiles down to javascript. It introduces type saftey, interfaces, easier to read syntax, and real time error detection.

Tools you will need.

  • Git Bash
  • VS Code
  • Node

Setup your project

  Create your root project folder and navigate to it with VS Code. Press CTRL + '~' To Open up the Terminal. Choose bash.exe from the dropdown.

Install Node dependencies

In the Terminal Window type the following commands.
  • $npm install -g typescript;
  • $npm install -g yarn
  • $npm install -g uglifyes;
To insure the tools installed correctly, run the following.
  • npm ls -g --depth 0;
