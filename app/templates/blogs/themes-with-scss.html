Whats SCSS?

Sassy CSS or SCSS is a pre-processor for CSS. We are going to use it to create a site that supports multiple themes. I think themes are a great addition to an website users will come back to everyday. Some websites you may only need to use once in a blue moon, you may not care as much how it looks. But for things like GMail and Twitter, themes make all of my apps match. I really enjoy a good dark theme. So any time a download a new app, the first thing I do is dig around in the settings, looking for a way to make it dark.
