Typescript with AngularJS and Node in VS Code.

By Alex Earley, Sunday November 19th, 2017

It took me the better part of yesterday to build this small little 'Hello World' app. I am glad that I did. AngularJS is my bread and butter at work, and my boss has shown interest in typescript. I wanted to introduce TypeScript to my stack, but not do it in the middle of a sprint at work. So I updated VS Code and installed TypeScript via NPM. I Google'd a handful of errors and questions, and soon enough I had a 'Hello World' alert popping up on ng-click.

Why TypeScript?

TypeScript is a superset of javascript. It compiles down to javascript. It introduces type saftey, interfaces, easier to read syntax, and real time error detection.

Tools you will need.

  • Git Bash
  • VS Code
  • Node

Setup your project

  • Create your root project folder and navigate to it with VS Code.
  • Create /app/
  • Create /app/app.ts
  • Create /index.html
  • Create /tsconfig.json
  • Press CTRL + '~' To Open up the Terminal. Choose bash.exe from the dropdown.

Install Node dependencies

In the Terminal Window type the following commands.
  • $npm install -g typescript;
  • $npm install -g yarn;
  • $npm install -g uglifyes;
To insure the tools installed correctly, run the following.
  • npm ls -g --depth 0;

Add JS dependencies with yarn.

The following commands will add library dependencies to node_modules folder.
  • $yarn add angular;
  • $yarn add angular-route;
  • $yarn add jquery;

Create a bundling script file

This is optional, you may run these commands however you'd like. I prefer to dump them into a bundle.sh file and run that.
  • Create bundle.sh in root directory, paste the following.
  • uglifyjs \
./node_modules/angular/angular.min.js \
./node_modules/angular-route/angular-route.min.js \
./app/app.js \ -o
./main.min.js \ --source-map url=main.min.js.map;

What do we have so far?

We have a bunch of empty files and a bundle.sh that does something. But no 'Hello World' yet. Now that we are almost done with set up, time to learn some fundamental things about TypeScript.
